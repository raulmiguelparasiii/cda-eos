# Prototype outputs

These files are generated outputs from the CDA-EOS numerical prototype.

- `stability_report.json`: base CoolProp/IAPWS-layer stability checks.
- `metric_curvature_report.json`: direct-metric Ruppeiner-curvature run summary.
- `association_sweep_summary.json`: parameter sweep summary for pressure-level CDA association correction.
- `cp_validation_report.json`: prototype Cp consistency/positivity check.
- `plot_tmd_comparison.png`: prototype alpha_p=0 diagnostic plot. This is not a fitted/converged TMD prediction and should not be used as a main paper figure.
